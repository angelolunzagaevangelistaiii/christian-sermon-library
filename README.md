# Christian Sermon Library (PHP + MySQL)

A searchable, bookmarkable sermon and Bible teaching library.

## Setup
1. Import `database/database.sql` into MySQL
2. Update `config/config.php` with your DB credentials
3. Place project inside XAMPP (htdocs) or Laragon www folder
4. Open `public/index.php` in browser

## Default Features
- Sermon search (AJAX)
- PDF download
- Video link
- Notes
- User Login/Register
